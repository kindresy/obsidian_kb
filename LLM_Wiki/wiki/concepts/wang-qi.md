# 王齐

《PCI Express 体系结构导读》作者，机械工业出版社 2010 年出版。

## Key Contributions

- 撰写《PCI Express 体系结构导读》— 以处理器体系结构为主线介绍 PCI 与 PCIe 总线的经典中文著作
- 侧重 [[host-bridge|HOST 主桥]] 和 [[root-complex|RC（Root Complex）]] 等 PCIe 规范未详细定义的实现细节
- 以 x86 和 [[powerpc|PowerPC]] 处理器为基础进行说明，涵盖 [[linux-pci-subsystem|Linux PCI 子系统]] 实践
- 第12章记录了 Capric 和 Cornus 卡的实际设计与 Linux 驱动开发经验
